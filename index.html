Correção do Erro de Permissão (RLS)

Se você vê o erro "RLS Policy" ou "Não permitiu gravar esta nova linha", siga estes passos no seu painel do Supabase:

  1. Abra o SQL Editor no menu lateral do Supabase.
  2. Clique em + New Query.
  3. Copie e cole o código abaixo e clique em RUN:
-- Liberar acesso para sincronia entre dispositivos
DROP POLICY IF EXISTS "Enable all for all" ON app_state;
CREATE POLICY "Enable all for all" ON app_state 
FOR ALL USING (true) WITH CHECK (true);
